Configuration requise
Avant la première utilisation :
  1. Démarrer Authentik : docker-compose up -d
  2. Accéder à Authentik : http://localhost:9090
  3. Configurer l'application OAuth2 :
    • Client ID : gyPvmpWdp...
    • Redirect URI : http://localhost:5000/auth/callback
    • Scopes : openid profile email groups
  4. Créer les groupes "Admins" et "read_users"
  5. Assigner les utilisateurs aux groupes appropriés
